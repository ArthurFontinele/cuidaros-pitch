# CuidarOS — Pitch Deck

A infraestrutura digital do cuidado domiciliar.

Pitch deck construído com [reveal.js](https://revealjs.com/), apresentado pela equipe da UNDB para o projeto PjBL de Banco de Dados (Economia do Cuidado).

## Apresentação ao vivo

👉 **[Abrir slides](https://arthurfontinele.github.io/cuidaros-pitch/)**

## Como apresentar

- Setas `←` `→` para navegar.
- `F` entra em tela cheia.
- `S` abre a tela do apresentador (com notas e cronômetro).
- `ESC` mostra a visão geral dos slides.
- `B` para tela preta (pausa).

## Conteúdo

10 slides cobrindo problema, solução, mercado, modelo de negócio, validação, time e ask.

## Rodar localmente

Não precisa instalar nada — abra o `index.html` no navegador. Para servir via http (recomendado, evita problemas de CORS):

```bash
python -m http.server 8000
```

Depois acesse `http://localhost:8000`.
