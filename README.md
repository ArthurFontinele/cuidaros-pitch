# EscalaBem — Pitch Deck

A infraestrutura digital do cuidado domiciliar.

Pitch deck construído com [reveal.js](https://revealjs.com/) para o projeto PjBL de Banco de Dados (Economia do Cuidado) — UNDB.

**Caso analisado:** CuidaBem (Marcela Viana), São Luís - MA — microempresa de cuidado domiciliar com 45 cuidadores e 90 famílias atendidas. Apresentação intermediada pela AM Consultoria (Adriano Matos).

## 🎬 Apresentação ao vivo

👉 **[Abrir slides](https://arthurfontinele.github.io/cuidaros-pitch/)**

## Como apresentar

| Tecla | Ação |
|---|---|
| `←` `→` | Navegar |
| `F` | Tela cheia |
| `S` | View do apresentador (com notas e cronômetro) |
| `ESC` | Overview |
| `B` | Blackout |

**Em celular:** deslize para os lados ou use os botões fixos no rodapé. Layout 100% responsivo.
